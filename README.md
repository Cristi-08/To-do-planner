# To-do Planner

Un organizator de sarcini smart cu citate motivaționale zilnice, categorii, priorități și statistici de productivitate.

## Cum se deschide

Deschide `index.html` în browser. Nu e nevoie de instalare.

## Tehnologii folosite

- HTML — structura paginii
- CSS — design, animații, teme dark/light
- JavaScript — logica aplicației
- API — citate motivaționale zilnice
- localStorage — salvarea taskurilor și preferințelor

## Funcționalități

- Adăugare task cu categorie și prioritate
- Bifare task completat
- Ștergere task
- Citat motivațional zilnic
- Temă dark/light cu salvare automată
- Statistici săptămânale de productivitate
